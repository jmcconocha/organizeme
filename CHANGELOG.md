## [1.0.1] - 2026-02-21

### Added
- **Dark Mode**: Switch between light and dark themes with an easy toggle
- **Project Dashboard**: Browse all projects with grid and table layout options
- **Sorting**: Organize projects by different criteria for quick discovery
- **Project Details**: View detailed information about each project including Git history and quick access to open in Finder
- **Quick Action Buttons**: Open projects directly in Terminal, VS Code, or GitHub
- **README Preview**: View project README files directly in the application
- **Improved Reliability**: Loading states and error handling for a smoother experience
- **Refresh Control**: Reload project information on demand

### Changed
- Dark mode background colors refined for improved visibility
- Application startup script enhanced for better process management

### Fixed
- Security vulnerabilities resolved through dependency updates
- Animation library import compatibility issue
- Startup script now properly handles hung processes and cache clearing

## [1.0.0] - 2026-02-20

### Added
- **Project Dashboard**: See all your development projects in one place with their current status and next steps. Quickly access your project files from the dashboard.
